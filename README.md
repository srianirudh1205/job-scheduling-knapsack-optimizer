# Job Scheduling + Knapsack Optimizer (C with DSA)

## 📌 Overview
This project showcases two classic optimization problems implemented in **C language with Data Structures and Algorithms (DSA)**:

- **Greedy Algorithm** → Job Scheduling to maximize profit before deadlines.  
- **Dynamic Programming (DP)** → 0/1 Knapsack Problem for resource allocation.  

It is designed to be **exam-ready, hackathon-friendly, and GitHub-submission compliant**.

---

## ⚙️ Features
- Job Scheduling using **Greedy approach** (profit-based sorting + slot allocation).  
- Knapsack Problem using **Dynamic Programming table**.  
- Clean modular C code with comments.  
- Demonstrates **structs, arrays, sorting, and DP tables**.  
- Ready to compile and run on any standard C compiler.
